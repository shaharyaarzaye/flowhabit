# FlowHabit 🚀

FlowHabit is a sleek and intuitive habit tracker designed to help you build positive routines, stay consistent, and achieve your goals. Whether you're tracking daily habits, setting reminders, or visualizing your progress, FlowHabit makes it easy to stay on track.

![FlowHabit Screenshot](https://via.placeholder.com/800x400.png?text=FlowHabit+Screenshot) <!-- Replace with actual screenshot -->

---

## Features ✨

- **Habit Tracking**: Log and monitor your daily habits effortlessly.
- **Progress Visualization**: View your streaks and progress with beautiful charts and graphs.
- **Reminders**: Set custom reminders to never miss a habit.
- **Goal Setting**: Define and achieve your personal goals.
- **Dark Mode**: A sleek dark theme for late-night habit tracking.
- **Cross-Platform**: Available on web, iOS, and Android.

---

## Technologies Used 🛠️

- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Mobile**: React Native
- **Authentication**: Firebase Auth
- **Deployment**: Vercel (Frontend), Heroku (Backend)
